Инcтрукция по запуску проекта локально

**Клонируем репозиторий c помощью команды**

```
 $ git clone https://vasilii-0009.github.io/moyoo/
```

**Устанавливаем зависимости**

```
$ npm i
```

**Запуск сервера в режиме разработки**

```
$ npm run dev
```

**Создание билда**

```
$ npm build
```

**Деплой проекта**

```
$ npm run deploy
```

## Стек:

- React
- TypeScript
- React-Hook-Form
